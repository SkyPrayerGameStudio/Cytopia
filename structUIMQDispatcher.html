Detailed Description

template<typename DataArgs>
class UIMQDispatcher< DataArgs >

An asynchronous dispatch to Observers based on a UIMessageQueue processed on another thread.

This dispatch policy allows for the GameLoop to asynchronously update the Views while modifying iModels
